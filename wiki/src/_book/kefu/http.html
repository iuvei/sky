1.1. getTalkingInfoNotLogin

ac getTalkingInfoNotLogin 动作标识

请求格式

{ "ac": "getTalkingInfoNotLogin", "client_type": 1, "edition": "v1.0.0" }

// 不可进入返回
{ "msg": 20030, "param": "聊天室禁止未登陆用户进入!!" }
